____ ИНСТРУКЦИЯ ПО УСТАНОВКЕ ____

1. Скачать файл kp(+26,27).local
   - Изменить название на  kp1.local
   - Добавить в корневую папку веб-сервера
   - Для OpenServer: domains/название_проекта/

3. СОЗДАНИЕ БАЗЫ ДАННЫХ
   - Запустить phpMyAdmin
   - Создать новую БД с именем 'pixel_art_db'
   - Импортировать файл dump01.sql (предварительно скачать из репозитория)

4. НАСТРОЙКА ПОДКЛЮЧЕНИЯ
   - Открыть файлы php
   - При необходимости изменить параметры:
     $host = '127.0.0.1';
     $dbname = 'pixel_art_db';
     $user = 'root';
     $pass = '';

5. ПРОВЕРКА РАБОТОСПОСОБНОСТИ
   - Главная страница: http://localhost/название_проекта/
   - Каталог товаров: http://localhost/название_проекта/patterns.php
   - Админ-панель: http://localhost/название_проекта/admin/
